Street Smartz ATL

A Google Maps Walking clone with APD crime data visualizations. This was for PantherHackers

sMarta

The purpose of the "sMarta" app was to make an application that is not only cool and creative, but most important useful. By turning human-like phrases into entities that can be read a variable to provide the correct information through API calls, we were able to create a google-home-based application that presents the user with accurate information on online rail-times and available amenities at all Marta Stations.

Street Smartz

With "Street Smartz", we wanted to create something not aggressively innovative, but progressively useful. With the global eco-friendly trend, the use of cars has digressed as time has progress. Just as time is recognized as a global entity, crime as well is a global occurence that can happen no matter where you are. We figured something useful to pedestrians around the world would be a system that could tell people what the most criminally populated areas were in their stroll.
